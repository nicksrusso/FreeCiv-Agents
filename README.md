# FreeCiv-Agents

This repo extends [CivRealm](https://github.com/bigai-ai/civrealm), a gym wrapper around [FreeCiv](https://github.com/freeciv). The goal is to train agents to play civ

## Getting Started

managed with poetry, run ```poetry install```

## Roadmap

1. Implement Single MCTS agent to play
2. Train distributed hierarchial agents to play
